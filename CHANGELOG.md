# Changelog

All notable changes to the Integrated Marketing Campaigns AI will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- Multi-provider LLM support (Anthropic, Google Gemini, OpenAI)
- 9-agent research and content generation pipeline
- Password-protected Streamlit UI with brand styling
- Web scraping with SSRF protection
- Document upload support (PDF, DOCX, TXT, MD)
- DOCX export for generated campaigns
- Campaign state export/import (JSON)
- Competitive analysis via Tavily integration
- Content quality review with automated refinement loop
- Rate limiting on LLM calls and web scraping
- Destructive action confirmation dialogs
- `.env.example` for developer onboarding
- Linter/formatter configuration (Ruff, Black)
